## Supported SQL Database Systems

- **MySQL**: 
  - MySQL Workbench
  - phpMyAdmin
  - CLI

- **PostgreSQL**:
  - pgAdmin
  - DBeaver
  - CLI

- **SQLite**:
  - DB Browser for SQLite
  - SQLite CLI

- **Microsoft SQL Server**:
  - SQL Server Management Studio
  - Azure Data Studio

- **Oracle Database**:
  - SQL Developer
  - SQL*Plus

- **MariaDB**:
  - MariaDB command line
  - phpMyAdmin
